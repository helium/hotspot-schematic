# Helium Hotspot Schematics

> [!NOTE]
> **⚠️ This repository is archived and read-only.** It is published for reference
> only. The design files are provided as-is, are not maintained, and no support
> is available. Issues and pull requests are not monitored.

Schematics for the **original Helium Hotspot**, the LoRaWAN gateway and
Proof-of-Coverage miner first sold by Helium in 2019.

The PDFs cover the Hotspot's concentrator sidecar: the carrier
board that sits alongside the Raspberry Pi via the 40-pin GPIO header and hosts a
mini-PCIe LoRa concentrator card alongside the Hotspot's supporting circuitry.
